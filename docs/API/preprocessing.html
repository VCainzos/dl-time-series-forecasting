Divides raw data into training and test set

  • train (float, optional) – training set threshold, defaults to 0.9

Returns

preprocessing.preprocessing.standarize(train_df, test_df)[source]

Centers data with mean 0 and standard deviation 1

\[\mathsf{on1} \times \mathsf{on2} = {\mathsf{normal} \over \left\| \mathsf{normal}\right\|}\]
Parameters

  • train_df (pandas.DataFrame) – training data

  • test_df (pandas.DataFrame) – test data

Returns

standarized training and test set

Return type

Note

Mean and standard deviation of the training set is used to center splits, in order to put test data information aside until the evalutation stage is done.
